Tema 4: Lenguaje DDL
Sentencia de creación: CREATE TABLE
Tipos de datos: varchar(n), char(n)... etc

Restricciones de tipo 1: una por columna. Cláusulas:
- NOT NULL
- PRIMARY KEY
- UNIQUE
- CHECK
- REFERENCES

Restricciones de tipo 2: una por una o varias columnas. Cláusulas:
- PRIMARY KEY
- FOREIGN KEY
- CONSTRAINT
- UNIQUE
